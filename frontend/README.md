# UI Service – PeerPrep

The **UI Service** is responsible for setting up the front end service of the web application

---

## Currently Implemented Features

- Login Page (Authenticate User Login)

# Setup

## 1. Install Node.js

Install Node.js (v24 recommended).

Verify installation:

```bash
node -v
npm -v
```

---

## 2. Install Dependencies

Navigate to the user-service folder:

```bash
cd .\frontend
npm install
```

---

## 3. Run

Navigate to the frontend folder:

```bash
cd .\frontend\peerprep-g15
npm install
npm run dev
```
